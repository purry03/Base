Base

Base is distractionless plaintext editor designed to quickly navigate between segments of an essay, or multiple documents. It features an auto-complete, synonyms suggestions, writing statistics, markup-based navigation and a speed-reader.

Install & Run

You can download builds for OSX, Windows and Linux, or if you wish to build it yourself, follow these steps: 

 git clone https://github.com/purry03/Base.git
 cd Base
 npm install
 npm start
