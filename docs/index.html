The minimalistic IDE

Base is open source. Be part of the Base community and help improve your favorite text editor.

What Base Offers

Minimal UI

Base sports a minimal UI design intended to reduce clutter and help you focus on your code while remaining aesthically pleasing.

Markdown Support

Base fully supports markdown syntax. Style your documents the way you like and export them. WYSIWYG!

Integrated Terminal

An Integrated Terminal mitigates window switching and speeds up your workflow.

In-Built Browser

Base features an inbuilt browser so you can access stack overflow without ever leaving the IDE.
